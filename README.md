# Better Choice

A calm, non-judgmental habit companion designed to support consistency without pressure, streaks, or guilt.

## 🌿 Overview

Better Choice explores an alternative approach to habit tracking.

Instead of using streaks, rewards, or notifications to drive behavior, this project focuses on creating a gentle and supportive experience that respects the user's pace.

The goal is to help users build habits in a way that feels sustainable, calm, and self-directed.

---

## 🎯 Problem

Many habit-tracking apps rely on:
- streaks that create pressure
- notifications that interrupt
- systems that can feel discouraging after missed days

These patterns can lead to guilt, anxiety, or abandonment of the habit altogether.

---

## 💡 Solution

Better Choice removes pressure-based mechanics and replaces them with:

- simple habit tracking (no streaks)
- gentle, supportive language
- flexible interaction without punishment

The experience focuses on:
> progress without pressure

---

## ✨ Key Features

- Add personal habits
- Mark progress with a simple “Done” toggle
- Gentle feedback instead of rewards or penalties
- Calm, minimal interface

---

## 🧠 Design Principles

- Non-judgmental interaction
- Emotional comfort over productivity pressure
- Simplicity over feature complexity
- User autonomy over system control
